# Windows
Module to work with Windows functions

## Commands
- Get Screen resolution
- Change resolution
- Get User Name
- Lock Windows Screen
- Is logged in
- Maximize window
- Restore window
- Minimize window
- List windows
- Set foreground window
- Find window
- Get service status
- Start service
- Stop service
